<code>❯

## :construction: Getting Started

## Project setup

```bash
$ yarn install
```

## Database configuration

Clone file:
```bash
.env.template
``` 
And rename to:
```bash
.env
``` 
```bash
# docker
docker-compose up -d
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
